## ส่วน ก — สร้าง OA ตั้งแต่ 0 (แท็บ「เริ่มจาก 0」)

ใช้เมื่อ**ยังไม่มี LINE OA** — ถ้ามีแล้วข้ามไปส่วน ข ได้เลย

### `oa-00-manager-login.png`

**ลำดับการกด / ทำอะไร**

1. เปิดเบราว์เซอร์ ไปที่ **https://manager.line.biz/**
2. กดปุ่มล็อกอิน (มักเป็น **Log in with LINE** / **ล็อกอินด้วย LINE**)
3. ยืนยันในแอป LINE หรือกรอกรหัสถ้าระบบขอ
4. **แคปหน้าจอตอนนี้** — ควรเห็นหน้าแรกหลังเข้า (รายการ OA หรือปุ่มสร้างใหม่)

---

### `oa-01-create-official-account.png`

**ลำดับการกด / ทำอะไร**

1. ในหน้า Manager หาปุ่มประมาณ **Create** / **สร้าง** / **เพิ่มบัญชีทางการ** / **Create new account**
2. เลือกประเภทร้าน/องค์กรตามที่ LINE ถาม (ถ้ามี)
3. กรอกชื่อร้าน อีเมล ข้อมูลที่ฟอร์มขอ
4. **แคปหน้าจอตอนกำลังกรอกหรือหน้าฟอร์มสร้าง OA** (ไม่ต้องแคปรหัสลับ)

---

### `oa-02-oa-home-dashboard.png`

**ลำดับการกด / ทำอะไร**

1. ทำขั้นตอนสร้างให้จบจนขึ้นว่าสำเร็จ
2. กดเข้าไปที่ **บัญชี OA ที่เพิ่งสร้าง** (คลิกชื่อร้าน / การ์ด OA)
3. **แคปหน้าจอแดชบอร์ดหลักของ OA** — เห็นเมนูซ้ายหรือปุ่มตั้งค่า (Friends, Chat, Settings ฯลฯ)

---

### `oa-03-developers-provider.png`

**ลำดับการกด / ทำอะไร**

1. เปิดแท็บใหม่ ไปที่ **https://developers.line.biz/console/**
2. กด **Log in** ด้วย LINE (แนะนำใช้บัญชี LINE **คนเดียวกับแอดมิน OA**)
3. ถ้ายังไม่มี Provider:
   - กด **Create** / **Create a new provider** / **สร้าง Provider**
   - ตั้งชื่อ (ชื่อร้านหรือทีม) → ยอมรับเงื่อนไข → กดสร้าง
4. ถ้ามี Provider แล้ว: กด **เลือก Provider** จากรายการ
5. **แคปหน้าจอ** — เห็นชื่อ Provider และพื้นที่ด้านล่าง (จะมีรายการ Channel หรือปุ่มเพิ่ม)

---

### `oa-04-channel-visible-in-console.png`

**ลำดับการกด / ทำอะไร**

1. อยู่ใน **Provider** เดียวกับขั้นก่อน
2. ถ้ายัง**ไม่มี Channel** ของ OA:
   - หาปุ่ม **Add** / **Create** / **Add a LINE Official Account** / **เพิ่มช่องทาง** แล้วทำตามจน OA ร้านคุณไปอยู่ในรายการ
   - หรือจากหน้า OA Manager บางเวอร์ชันมีลิงก์ให้ “เชื่อม” กับ Developers — ทำตามจน Channel โผล่ใน Console
3. **แคปหน้าจอ** — เห็น**แถว/การ์ด Channel** ที่เป็นชื่อ OA ร้านคุณ (อาจมีไอคอน Messaging API)

---

## ส่วน ข — เชื่อมต่อ JongMe (ทุกคน)

ทำใน **LINE Developers** + หน้า **ตั้งค่า JongMe**

### `01-console-home.png`

**ลำดับการกด**

1. **https://developers.line.biz/console/** → ล็อกอิน
2. คลิก **Provider** ที่มี OA ร้านคุณ
3. **แคป** — เห็นหน้ารวมของ Provider (มุมที่เห็นชื่อ Provider ชัด)

---

### `02-channel-list.png`

**ลำดับการกด**

1. ในหน้า Provider เลื่อนดูรายการ **Channels**
2. **คลิกเลือก Channel** ที่เป็น OA ร้าน (ไม่ใช่แบบ LINE Login อย่างเดียว)
3. **แคป** — เห็นรายการหลาย channel หรือหน้าหลังคลิกเข้า channel แล้วเห็นชื่อ channel ชัด

---

### `03-open-messaging-api-tab.png`

**ลำดับการกด**

1. หลังเข้า **Channel** ของ OA แล้ว ดู**แท็บด้านบน**
2. ยัง**ไม่ต้องกด** — หรือกดไปที่แท็บ **Messaging API** ก็ได้
3. **แคป** — เห็นแท็บ **Basic settings**, **Messaging API**, **LIFF** (หรือคล้ายกัน) ในแถวเดียวกัน

---

### `04-basic-settings-secret.png`

**ลำดับการกด**

1. คลิกแท็บ **Basic settings** / **การตั้งค่าพื้นฐาน**
2. เลื่อนหา **Channel secret**
3. กด **Show** / **แสดง** / ไอคอนตา (ถ้ามี) เพื่อให้เห็นค่า — **แคปแล้วควรทับค่าจริง** หรือแคปแค่ตำแหน่งช่อง
4. คัดลอก secret ไปวางใน JongMe (นอกรูปได้)

---

### `05-messaging-api-webhook.png`

**ลำดับการกด**

1. คลิกแท็บ **Messaging API**
2. เลื่อนหา **Webhook settings**
3. เปิดสวิตช์ **Use webhook** = **เปิด (ON)**
4. วาง **Webhook URL** จากหน้าตั้งค่า JongMe (กดคัดลอกใน JongMe)
5. กด **Update** / บันทึก ถ้ามี
6. กด **Verify**
7. **แคป** — เห็นช่อง URL + สวิตช์ + ปุ่ม Verify (URL โบ๋ได้)

---

### `06-messaging-api-token.png`

**ลำดับการกด**

1. ยังอยู่แท็บ **Messaging API**
2. เลื่อนหา **Channel access token** แบบ **long-lived** (ไม่ใช่ short-lived)
3. กด **Issue** / **Generate** / **สร้าง token**
4. คัดลอก token ยาว ๆ ไปวาง JongMe
5. **แคป** — เห็นบล็อก token (ทับตัวอักษรจริง) + ปุ่ม Issue

---

### `07-jongme-settings-line.png`

**ลำดับการกด**

1. เปิดเว็บ JongMe → เข้าสู่ระบบร้าน → **ตั้งค่า**
2. เลื่อนถึงหัวข้อ **LINE OA ตั้งค่า**
3. **แคป** — เห็นช่อง Webhook URL, Channel Access Token, Channel Secret, ปุ่มทดสอบ (ทับข้อมูลลับ)
